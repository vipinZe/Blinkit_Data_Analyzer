# 🛒 Blinkit Data Analyzer

Blinkit Data Analyzer is an interactive web application built using Streamlit that helps analyze and visualize grocery sales data. The app provides insights into product performance, pricing trends, and category-wise analysis using a real-world dataset.

---

## 🚀 Features

- 📊 Interactive data visualization
- 🛍️ Category-wise product analysis
- 💰 Price and sales insights
- 📈 Trend analysis using charts
- 🔍 Easy-to-use dashboard interface

---

## 🧰 Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib / Seaborn / Plotly

---

## 📂 Project Structure

Blinkit_Data_Analyzer/
│
├── streamlit_app.py # Main application
├── blinkit_data.csv # Dataset
├── requirements.txt # Dependencies
└── data_generator.py # Data preprocessing script

---

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/vipinZe/Blinkit_Data_Analyzer.git
cd Blinkit_Data_Analyzer
````
2. Install dependencies:
```bash
pip install -r requirements.txt
```
3. Run the app:
```bash
streamlit run streamlit_app.py
```

---

🌐 Deployment

This project can be deployed on platforms like Render.

Start Command:
```bash
streamlit run streamlit_app.py --server.port 10000 --server.address 0.0.0.0
```
