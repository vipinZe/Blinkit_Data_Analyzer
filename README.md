📊 Blinkit Data Analysis Dashboard:
An interactive data analysis dashboard built using Streamlit, Python, Pandas, and Plotly to explore and visualize Blinkit dataset insights.

🚀 Live Demo:
🔗 Add your deployed link here (after deploying on Streamlit Cloud)

📌 Features:
📈 Interactive data visualizations using Plotly
🔍 Dynamic filtering and exploration of dataset
💡 Key insights on ratings, cost, and restaurant performance
📊 Scatter plots, trends, and KPI metrics
⚡ Fast and user-friendly interface with Streamlit

🛠️ Tech Stack:
Python
Streamlit
Pandas
NumPy
Matplotlib
Plotly
